# 매뉴얼 안내 (시작 페이지)

이 폴더는 **Kiki Trade AI**를 설치·사용·운영할 때 보는 안내서입니다.  
개발자가 아니어도 따라 할 수 있도록, **실제 사용하는 순서**로 정리했습니다.

> 화면에 없는 고급 API는 [API사용매뉴얼.md](API사용매뉴얼.md) · OpenAPI `/docs` 를 사용하세요.  
> 문서 CMS: Admin `/admin/docs` (저장소 Markdown 읽기 전용).

개발·설계용 문서(더 기술적인 내용): [../README.md](../README.md)

---

## 용어를 먼저 알아두기

| 쉬운 말 | 전문 용어 | 뜻 |
|---------|-----------|-----|
| 관리 화면 | Admin / Frontend | 브라우저로 여는 관리자 웹 (`http://localhost:3000`) |
| 서버 / 백엔드 | Backend / API | 실제 일을 처리하는 프로그램 (`http://127.0.0.1:8000`) |
| 데이터베이스 | DB / PostgreSQL | 시세·주문·설정을 저장하는 창고 |
| AI 엔진 | Ollama / LLM | PC에서 돌아가는 인공지능 프로그램 |
| 건강 점검 | Health (`/health`) | 서버·DB·AI가 살아 있는지 확인하는 주소 |
| API 설명서 | OpenAPI / Swagger (`/docs`) | 버튼으로 API를 시험해 보는 웹 페이지 |
| 모의 | Mock / Paper | 실제 돈이 나가지 않는 연습 모드 |
| 실전 | Live | 실제 계좌로 주문이 나가는 모드 (기본 **꺼짐**) |
| 비상 정지 | Kill Switch | 주문을 긴급히 막는 스위치 |
