## 설치 순서

### 1. Clone

```powershell
git clone https://github.com/kihwan1017-prog/stock.git
cd stock
# 또는 기존 경로: D:\Projects\stock-platform
```

### 2. Python 가상환경

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
```

### 3. 환경변수

1. `.env.example`을 참고해 시크릿 파일 작성  
2. 기본 로드 경로(코드): `E:\StockTrading\secrets\stock-platform.env`  
3. 최소 설정: `DB_*`, (선택) `KIWOOM_*`, `OLLAMA_*`, `ADMIN_API_KEY`

상세: [../deployment/CONFIGURATION.md](../deployment/CONFIGURATION.md)

권장 초기값:

```dotenv
KIWOOM_USE_MOCK=true
KIWOOM_LIVE_ORDER_ENABLED=false
SCHEDULER_ENABLED=true
ADMIN_API_KEY=
```

### 4. PostgreSQL DB 생성

PostgreSQL에서 DB/유저를 만들고 `.env`의 `DB_NAME`/`DB_USER`/`DB_PASSWORD`와 맞춥니다.  
연결 테스트:

```powershell
.\.venv\Scripts\python.exe scripts\test_db_connection.py
```

### 5. Alembic Migration

```powershell
$env:PYTHONPATH = "D:\Projects\stock-platform\src"
.\.venv\Scripts\python.exe -m alembic upgrade head
.\.venv\Scripts\python.exe -m alembic heads
```

- 설정: `alembic.ini` → `script_location = database/alembic`
- 검증: `.\scripts\verify_alembic.ps1`

### 6. Backend 실행

```powershell
$env:PYTHONPATH = "D:\Projects\stock-platform\src"
.\.venv\Scripts\uvicorn.exe stock_platform.api.main:app --host 127.0.0.1 --port 8000
```

확인:

- http://127.0.0.1:8000/health
- http://127.0.0.1:8000/docs

## Frontend 설치

```powershell
cd D:\Projects\stock-platform\frontend
npm install
copy .env.example .env.local   # 필요 시 수정
npm run dev
```

기본 `.env.local`:

```dotenv
NEXT_PUBLIC_API_BASE_URL=http://127.0.0.1:8000
NEXT_PUBLIC_API_PREFIX=/api/v1
NEXT_PUBLIC_AUTH_MODE=disabled
```

접속: http://localhost:3000

상세: [../../frontend/README.md](../../frontend/README.md)

## 초기 계정

| 구분 | 상태 |
|------|------|
| Admin 웹 JWT 계정 | **없음** |
| `AUTH_MODE=disabled` | 로그인 화면에서 **개발 모드로 입장** (로컬 세션) |
| `ADMIN_API_KEY` | 비어 있으면 관리 API 로컬 통과; 운영에서는 반드시 설정 |
| DB 사용자 테이블 로그인 | Admin UI에 미구현 |

> 스크린샷 권장: 설치 후 `/health` JSON, `/docs` Swagger, Admin 로그인 화면
