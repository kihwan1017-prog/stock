## 프로그램 시작

### Backend (필수)

```powershell
cd D:\Projects\stock-platform
.\.venv\Scripts\Activate.ps1
$env:PYTHONPATH = "D:\Projects\stock-platform\src"
.\.venv\Scripts\uvicorn.exe stock_platform.api.main:app --host 127.0.0.1 --port 8000
```

기동 시 lifespan이 DB 점검, (설정 시) broker recovery·내부 스케줄러·시세 영속화 등을 시도합니다.

### Admin Frontend (선택)

```powershell
cd D:\Projects\stock-platform\frontend
npm run dev
```

## Scheduler

### API lifespan 내부 스케줄러

uvicorn 기동 시 함께 올라가는 작업(예: daily loss, strategy runtime reload 등).  
별도 프로세스가 아니라 **API 프로세스와 함께** 종료됩니다.

### AutomaticScheduler (`scripts/run_scheduler.py`)

`SCHEDULER_ENABLED=true`일 때만 기동합니다.

```powershell
$env:PYTHONPATH = "D:\Projects\stock-platform\src"
.\.venv\Scripts\python.exe scripts\run_scheduler.py
```

등록 Job 예:

- `candidate_screening`
- `ai_orchestration`
- `position_planning`
- (레지스트리) `daily_operations_report`, `upbit_krw_daily_sync`, `indicator_daily_batch`

수동 1회:

```powershell
.\.venv\Scripts\python.exe scripts\run_scheduler_job.py candidate_screening
```

관리 API (Admin Key 필요, 키가 비어 있으면 로컬 통과):

```http
POST /api/v1/scheduler-admin/run-now/{job_name}
X-Admin-API-Key: <ADMIN_API_KEY>
```

Windows 작업 등록/해제:

- `scripts/install_scheduler_task.ps1`
- `scripts/uninstall_scheduler_task.ps1`

### Scheduler 종료

- `run_scheduler.py` 프로세스 `Ctrl+C`
- 또는 uninstall 스크립트 / 작업 스케줄러 중지

## 자동매매(실전/모의)

| 항목 | 구현 |
|------|------|
| 실전 주문 | `KIWOOM_LIVE_ORDER_ENABLED` (기본 `false`) |
| 모의 키움 | `KIWOOM_USE_MOCK=true` |
| Paper API | `/api/v1/paper-orders` 등 (OpenAPI 참고) |
| Admin UI에서 시작/중지 버튼 | **미구현** (Coming Soon) |

실전 전환 API prefix: `/api/v1/broker/live-transition`  
Kill Switch: `/api/v1/risk/kill-switch` (activate/deactivate는 Admin Key)

**운영 기본:** 실전 주문 플래그를 켜지 않습니다.  
체크리스트: [../trading/LIVE_TRADING_CHECKLIST.md](../trading/LIVE_TRADING_CHECKLIST.md)

## 로그 확인

- uvicorn/스케줄러를 띄운 **터미널 표준 출력** (전용 로그 파일 로테이션 UI 없음)
- 감사 로그 API: `GET /api/v1/audit/events` (Admin Key)
- Admin Dashboard `recent_errors`는 API `/api/v1/system/dashboard` 응답에 포함 (Admin UI는 health 위주)
