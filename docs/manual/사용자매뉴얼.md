## 프로그램 소개

Kiki Trade AI는 키움(주식)·업비트(암호화폐) 데이터를 바탕으로 후보 선정·AI 분석·리스크·주문 파이프라인을 제공하는 **자동매매 플랫폼**입니다.

- **Backend:** FastAPI (`http://127.0.0.1:8000`)
- **Admin:** Next.js (`http://localhost:3000`)
- **DB:** PostgreSQL
- **LLM:** Ollama (로컬)

실전 주문은 기본 **차단**입니다 (`KIWOOM_LIVE_ORDER_ENABLED=false`).

## 로그인

| 항목 | 구현 상태 |
|------|-----------|
| JWT / 비밀번호 로그인 | **미구현** |
| `NEXT_PUBLIC_AUTH_MODE=disabled` | 로그인 화면에서 **「개발 모드로 입장」** → 로컬 세션 후 Dashboard 이동 |

절차:

1. Backend 실행 확인 (`/health`)
2. `http://localhost:3000` 접속 (미인증 시 `/login`으로 이동)
3. **개발 모드로 입장** 클릭

> 📷 스크린샷: 로그인 화면 · 「개발 모드로 입장」 버튼
